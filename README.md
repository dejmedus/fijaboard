## Fijabord 


### Run locally

#### Backend

```zsh
cd backend
source venv/bin/activate
python3 app.py
```
#### to test backend Flask models in the flask shell:
``` 
flask shell
exit()
```
#### Frontend
```zsh
cd frontend
npm run dev
```

### Setup

#### Backend

```zsh
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

#### Frontend

```zsh
cd frontend
npm install
```
