## Fijabord 


### Run locally

#### Backend

```zsh
cd backend
source venv/bin/activate
```

#### Frontend
```zsh
cd frontend
npm run dev
```

### Setup

#### Backend

```zsh
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

#### Frontend

```zsh
cd frontend
npm install
```
